## 这是什么

这是《Casualties Unknown》的一款个人定制版简体中文本地化。

英文原文是冷静的说明书。准确，详尽，但不负责教你活下去。你得自己试过才知道它真的会杀了你。

游戏已有的那份社区精准翻译严谨、忠实、克制，


本版本不试图取代它，**而是走另一条路：**

本版本是一份**活下来的队友写的笔记**。不一定每个词都对应原文，但会告诉你「这一步可能死」。

“不忠实”的翻译不等于劣质的翻译。这里的「不忠实」是故意的，是设计选择。

---

## 和社区精准版的区别

| | 社区精准版 | tigerLin定制版 |
|---|---|---|
| 追求 | 准确还原原文 | 优化游玩体验 |
| 语言风格 | 严谨、中性 | 口语化、有梗 |
| 适合谁 | 想读原文想表达的意思 | 想快速理解机制、沉浸氛围 |
| 额外内容 | 无 | 新手提示、机制讲解、幽默吐槽 |

打个比方。`adhesivebandage`（创可贴）这条：

> **英文原文：** "A pack of small medical dressings, made to treat small injuries. Lowers bleeding."
>
> **社区精准版：** 一包小型医用敷料，用于处理轻微伤口。可减少出血。
>
> **本版本：** "一包小型医用敷料，用于处理轻微伤口。可减少出血。
> 
> <color=yellow>如果你有非常着急的大出血，可以快速使用大量创可贴来止血</color>",

三句话，三种气质。选你需要的那个


本版本特别强调「新手友好」：
- 危险机制用红色警告醒目标注
- 实用技巧用黄色提示高亮

## 本版本做了什么

**硬核医疗，软性讲解**

游戏拥有极其复杂的医疗系统（骨折、出血、感染、截肢、阿片类药物过量……）。

本翻译尽量把每种药品的实际用途、风险和注意事项讲得通俗易懂，而不是照搬原文

简而言之

**不需要自己试错才知道什么会要命**

**重建了信息层级**

- 红色：致命警告，认真看
- 黄色：实用提示，建议看
- 橙色：轻松吐槽，随便看
- 灰色：彩蛋

**给了游戏一个叙事声音。**

原版像监控摄像头：客观，有距离。本版本像一个跟你一起被扔进洞穴的倒霉蛋在物品上贴的便条————懂的不一定多，但吃过亏是真的。

**中文互联网语境**

部分描述使用了中文网络文化中的表达。如果你看到看不懂的梗可以去去B站补课。

## 常见问题

**Q: 为什么叫「不那么忠于原文」？这不是翻译错误吗？**

不是错误，是设计。本版本的核心理念是在理解原文含义的基础上，用中文玩家更亲切、更好记的方式表达。如果你追求每个词都严格对应原文，社区精准版更适合你。

**Q: 里面的梗会不会过时？**

会。网络梗有保质期，这是本版本的一部分。把它当作2026年的时代胶囊，而不是永恒文本。

后期如果有机会的话会尽量更新

**Q: 多人模式能用吗？**

理论上能，但未经测试。如果你在多人模式中使用了，欢迎反馈。

**Q: 发现有翻译问题或想提建议？**

去B站找 @_tigerLin_ 反馈，提 Issue 嘛...我好像不常上GitHub，因此随缘回复

另外，迫于学业压力，仅周末可以回复
